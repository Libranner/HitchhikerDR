h1. Hitchhiker

This a web based application where people can create trips and find other people to share the costs. I used Ruby on Rails, MongoID and bootstrap to developed this project, feel free to fork it.

h2. Ruby on Rails

This application requires:

* Ruby version 1.9.3
* Rails version 3.2.13

Learn more about "Installing Rails":http://railsapps.github.io/installing-rails.html.

h2. Database

This application uses MongoDB with the Mongoid ORM.

h2. Development

* Template Engine: Haml
* Testing Framework: RSpec and Factory Girl and Cucumber
* Front-end Framework: Bootstrap-rails
* Form Builder: Simple Form
* Authentication: Devise
* Authorization: Can Can

h2. Email

The application is configured to send email using a Gmail account.
